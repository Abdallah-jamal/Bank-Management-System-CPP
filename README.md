# Bank Management System (C++)

Console-based Bank Management System built using C++ and Object-Oriented Programming.

## Features
- Client management (Add, Update, Delete, Find, List)
- User management with login system
- Currency management and exchange rate calculation
- Deposit, Withdraw, and Transfer operations
- Transfer log and login register
- File handling as back-end storage
- Clear separation between UI (Screens) and Business Logic

## Technologies
- C++
- Object-Oriented Programming (OOP)
- File Handling
